# ToDoList App (React + Vite) 
aplicacion de gestion de tareas desarrollada con React y Vite. 
permite añadir, completar y eliminar tareas de una forma sencilla y rápida.
Proyecto creado para practicar fundamentos de React con componentes funcionales y hooks.

## Tecnologías utilizadas:
- REACT
- VITE
- JAVASCRIPT (ES6+)
- HOOKS (useState)
- CSS modules / estilos de linea

## Funcionalidades 
- añadir tareas
- marcar tarea completada
- eliminar tarea
- contador dinamico de tarea


## Que aprendí? 
- Manejo del estado con UseState
- Randerizado condicional
- componentizacion en React
- Gestion de eventos en JSX
- Uso de Vite como entorno rapido de desarrollo.

  Proyecto creado con fines educativos.
